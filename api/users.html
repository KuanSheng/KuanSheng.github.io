1.2 用户管理

1. 个人中心

1.1 功能描述

获取个人中心。

1.2 请求说明

请求方式：GET
请求URL ：/api/personal

1.3 请求参数

无参数

1.4 返回结果

{
    "user_id": 1,
    "last_name": "Kris",
    "first_name": "Sheng",
    "name": "Kris Sheng",
    "email": "admin@example.com",
    "headline": "Data Scientist",
    "phone_number": "12345678",
    "created_at": "2013-01-29T12:34:56.000000Z",
    "updated_at": "2013-01-29T12:34:56.000000Z"
}

1.5 返回参数

字段 字段类型 字段说明
user_id string id
name string 姓名
email string 邮箱
phone string 电话
verifiedStatus string 是否实名制认证，"已认证"，"未认证"，"处理中"

每组数组用空行隔开

1.6 错误状态码

参见 全局响应状态码说明
