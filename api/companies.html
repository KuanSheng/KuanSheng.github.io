1.3 企业管理

1. 企业

1.1 功能描述

获取企业详情

1.2 请求说明

请求方式：GET
请求URL ：api/companies/:companyid/

1.3 请求参数

无参数

1.4 返回结果

{
    "company_id": 1,
    "company_name": "aipin job technology",
    "website": "www.example.com",
    "phone_number": "12345678",
    "email": "example@gmail.com",
    "company_profile": "This is a great company."
}

1.5 返回参数

字段 字段类型 字段说明
company_name string 企业名.
website string 企业官网地址.
phone_number (optional) string 联系电话.
company_profile (optional) string 企业描述.
email string 企业邮箱地址.

每组数组用空行隔开

1.6 错误状态码

参见 全局响应状态码说明

2. 创建企业

2.1 功能描述

创建企业

2.2 请求说明

请求方式：POST
请求URL ：api/companies/

2.3 请求参数

字段 字段类型 字段说明
company_name string 企业名.
website string 企业官网地址.
certification_number string 营业执照号码.
phone_number (optional) string 联系电话.
company_profile (optional) string 企业描述.
email string 企业邮箱.
password string 企业登录密码.

2.4 返回结果

201 created

{
    "company_id":"1"
}

400 bad request

{
    "description":"企业存在"
}

1.5 返回参数

字段 字段类型 字段说明
job_id string 职位id

每组数组用空行隔开

1.6 错误状态码

参见 全局响应状态码说明
