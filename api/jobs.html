1.4 职位管理

1. 职位

1.1 功能描述

获取职位列表(小程序企业端)

1.2 请求说明

请求方式：GET
请求URL ：api/companies/:company_id/jobs?mobile=true

1.3 请求参数

无参数

1.4 返回结果

[
    {
        "job_id": 1,
        "job_title": "Data Scientist",
        "department": "Data",
        "favorite": true,
        "city":"上海",
        "archived": false,
        "last_candidate_updated_at": "2013-01-29T12:34:56.000000Z"
    }
]

1.5 返回参数

字段 字段类型 字段说明
job_title string 职位名称
department (optional) string 部门
city string 城市
last_candidate_updated_at string 创建时间

每组数组用空行隔开

1.6 错误状态码

参见 全局响应状态码说明

2. 职位详情

2.1 功能描述

获取职位详情

2.2 请求说明

请求方式：GET
请求URL ：api/public/jobs/:id/

2.3 请求参数

无参数

2.4 返回结果

{
    "job_title": "Data Scientist",
    "department": "Data Science Offers",
    "internal_code": "123",
    "country": "China",
    "city": "Beijing",
    "zip": "100029",
    "is_remote": true,
    "description": "This is job description",
    "requirements": "This is job requirements",
    "benefits": "This is job benefits",
    "company_industry": "IT",
    "job_function": "Technology Support",
    "employment_type": "Full Time",
    "experience": "Associate",
    "education": "Master",
    "keywords": ["Marketing", "Data"],
    "salary_min": 1000,
    "salary_max": 20000,
    "salary_currency": "USD",
    "referral_email": "example@gmail.com"
}

2.5 返回参数

字段 字段类型 字段说明
job_title string 职位详情.
department (optional) string 部门
internal_code (optional) int 内部代码
country (optional) string 国家
city (optional) string 城市
is_remote (optional) bool 是否是远程办公
description (optional) string 职位描述
requirements (optional) string 职位要求
benefits (optional) string 职位福利
company_industry (optional) string 公司行业
employment_type (optional) string 全职，兼职或实习
experience (optional) string 经验要求 例如："3-5年"
education (optional) string 教育背景要求
keywords (optional) array[string] 关键词
salary_min (optional) int 月薪下限
salary_max (optional) int 月薪上限

每组数组用空行隔开

2.6 错误状态码

参见 全局响应状态码说明

3. 创建职位（小程序端）

3.1 功能描述

创建职位并发布

3.2 请求说明

请求方式：POST
请求URL ：api/companies/:companyId/jobs/client/

3.3 请求参数

字段 字段类型 字段说明
job_title string 职位详情.
city string 城市
description string 职位描述
requirements string 职位要求
company_industry (optional) string 公司行业
employment_type (optional) string 全职，兼职或实习
experience (optional) string 经验要求 例如："3-5年"
education (optional) string 教育背景要求
salary_min (optional) int 月薪下限
salary_max (optional) int 月薪上限

3.4 返回结果

{
    "job_id": "1"
}

3.5 返回参数

201 created

每组数组用空行隔开

3.6 错误状态码

参见 全局响应状态码说明

4. 归档职位

4.1 功能描述

归档后的职位将不在小程序端显示

4.2 请求说明

请求方式：PUT
请求URL ：api/companies/:companyId/jobs/archive_status/

4.3 请求参数

{
    "archived": true
}

4.4 返回结果

{
    "job_id": "1"
}

4.5 返回参数

字段 字段类型 字段说明
job_id string 职位id

4.6 错误状态码

参见 全局响应状态码说明
