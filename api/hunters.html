1.5 猎头管理

1. 猎头

1.1 功能描述

获取猎头列表

1.2 请求说明

请求方式：GET
请求URL ：api/hunters/

1.3 请求参数

无参数

1.4 返回结果

[
    {
        "name":"Kris",
        "subscription_fee":123
    }
]

1.5 返回参数

字段 字段类型 字段说明
name string 企业名.
subscription_fee number 订阅费.

每组数组用空行隔开

1.6 错误状态码

参见 全局响应状态码说明

2. 猎头简历列表

2.1 功能描述

获取猎头简历列表

2.2 请求说明

请求方式：GET
请求URL ：api/hunters/:id/resumes/

2.3 请求参数

无参数

2.4 返回结果

[
    {
        "name":"Kris",
        "last_school":"北京大学"
    }
]

400 bad request

{
    "description":"企业存在"
}

1.5 返回参数

字段 字段类型 字段说明
name string 姓名
last_school string 毕业院校
last_position string 最近职位
expect_area string 期望工作地点
expect_salary string 期望薪资

每组数组用空行隔开

1.6 错误状态码

参见 全局响应状态码说明

3. 订阅猎头

3.1 功能描述

订阅猎头

3.2 请求说明

请求方式：PUT
请求URL ：api/hunters/:id/subscription/

3.3 请求参数

字段 字段类型 字段说明
subscription bool 订阅.

3.4 返回结果

{
    "hunter_id":"1"
}

3.5 返回参数

字段 字段类型 字段说明
hunter_id string 猎头id

每组数组用空行隔开

3.6 错误状态码

参见 全局响应状态码说明

4. 购买简历

4.1 功能描述

购买简历

4.2 请求说明

请求方式：PUT
请求URL ：api/companies/:comapanyId/

4.3 请求参数

字段 字段类型 字段说明
company_name string 企业名.
website string 企业官网地址.
certification_number string 营业执照号码.
phone_number (optional) string 联系电话.
company_profile (optional) string 企业描述.
email string 企业邮箱.

4.4 返回结果

201 created

{
    "company_id":"1"
}

4.5 返回参数

字段 字段类型 字段说明
company_id string 企业id

每组数组用空行隔开

4.6 错误状态码

参见 全局响应状态码说明
